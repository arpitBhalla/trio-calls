## ğŸš€ Getting started

Guide for local deployment -

- Clone the repository

```bash
git clone git@github.com:arpitBhalla/MicrosoftEngageMentorship2021.git
```

- Install dependencies

```bash
npm install
```

- Start MongoDB

```bash
mongod
```

- Run the app

```bash
npm start:client
```

```bash
npm start:server
```

- Open project at `http://localhost:3000`
